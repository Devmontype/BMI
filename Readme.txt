To run this project use below procedure

1.Run npm install
2.Create a file with ".json" extension
    e.g : abc.json
3.Add some content like "people.json" file in the newly created file
    e.g:[{
        "Gender": "Male",
        "HeightCm": 171,
        "WeightKg": 96
    }]
4.The run the command in terminal npm start abc.json
   *Note : abc.json is newly created file name
